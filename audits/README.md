# Audit Reports

This directory contains audit reports generated by the `/audit` command.

## How to Use

1. In Cursor, select files or folders using `@filename` or `@foldername`
2. Type `/audit` in the chat
3. The audit will be performed and saved here automatically

## File Naming

- Single file: `audits/<filename>-audit.md`
- Folder: `audits/<folder-name>-audit.md`
- Entire codebase: `audits/codebase-audit.md`
